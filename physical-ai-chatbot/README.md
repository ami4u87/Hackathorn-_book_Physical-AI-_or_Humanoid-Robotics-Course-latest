# Physical AI Course Chatbot

An AI-powered chatbot trained on the complete Physical AI and Humanoid Robotics course content.

## Features
- RAG (Retrieval-Augmented Generation) with Qdrant vector database
- Powered by Google Gemini 1.5 Flash
- Trained on 659 course content chunks
- Covers ROS 2, simulation, perception, and VLA integration

## Try it!
Ask questions about robotics, ROS 2, simulation environments, and more!

## Topics Covered
- **ROS 2 Fundamentals**: Nodes, topics, services, actions, parameters
- **Simulation Environments**: Gazebo Harmonic, Unity, NVIDIA Isaac Sim
- **Perception & Control**: Computer vision, object detection, pose estimation
- **Vision-Language-Action**: GPT-4 Vision integration, multi-modal processing
- **Capstone Projects**: System integration and autonomous robotics
