# Module 1: ROS 2 Fundamentals

This module introduces the core concepts of ROS 2, including the publisher-subscriber pattern, services, actions, parameters, and launch files. By the end of this module, you will understand how to create basic ROS 2 nodes and communicate between them.

## Learning Objectives

- Understand the ROS 2 architecture and concepts
- Create publisher and subscriber nodes
- Implement services and actions
- Use parameters and launch files
- Work with TF2 transformations
- Complete hands-on exercises to reinforce learning
